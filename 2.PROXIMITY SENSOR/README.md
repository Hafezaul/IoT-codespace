<h1 center="align">Sensor Jarak</h1>
<img src="sensor.png">
<p>Pengukur jarak dengan sensor ultrasonic HC-SR04 dan Arduino uno</p>

<h3>Komponen yang Digunakan:</h3>

<ul>
<li>Arduino Uno</li>
<li>Sensor ultrasonic HC-SR04</li>
<li>Kabel jumper</li>
</ul>

<h3>Koneksi kabel</h3>

| HC-SR04 | Arduino Uno |
| ------- | ----------- |
| VCC     | 5V          |
| GND     | GND         |
| Trig    | Pin 9       |
| Echo    | Pin 10      |
